# 🧪 Software Project Testing

A collection of Java-based software testing projects covering **unit testing**, **integration testing**, and **UI automation with Selenium**. Each project demonstrates real-world testing strategies applied to common software systems.

---

## 📁 Projects

### 🏦 [Banking System](./Banking-System/Banking%20System)
Testing core banking operations including account creation, deposits, withdrawals, and balance checks. Covers both positive and negative test cases to ensure reliability and edge case handling.

### 🧮 [Calculator](./Calculator%20Testing%20Project/Calculator)
Unit tests for arithmetic operations (addition, subtraction, multiplication, division). Includes boundary value analysis and equivalence partitioning test cases.

### 🎓 [Student Management System](./Student-Management-System)
Testing CRUD operations for managing student records — add, update, delete, and search functionality verified through structured test suites.

📄 **[Selenium Test Report](./Selenium%20Test.pdf)** — Full UI automation test results and analysis

## ✅ Testing Approaches Used

- **Unit Testing** — Testing individual methods and functions in isolation
- **Integration Testing** — Verifying interaction between system components
- **UI Automation** — Selenium-based end-to-end browser testing
- **Boundary Value Analysis** — Edge case coverage for inputs
- **Equivalence Partitioning** — Grouping inputs into valid/invalid classes
